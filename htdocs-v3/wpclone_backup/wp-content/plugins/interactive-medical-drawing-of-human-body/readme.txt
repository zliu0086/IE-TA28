Interactive medical diagram of human body anatomy with clickable body parts including head, chest, abdomen, etc

== Description ==
**[LIVE DEMO](https://www.humananatomyillustrations.com/interactive-human-body-wordpress-plugin.html)**

This interactive medical diagram of human body anatomy displays 28 clickable body parts/organs. It gives you the option to customize the links, hover description for each organ/body part through an easy to use visual editor at the dashboard without any coding knowledge, then insert the diagram shortcode `[anatomy]` in any page/post to display the interactive drawing.

== Installation ==

1. Install the plugin .zip file through your WordPress 'Plugins' screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Now, you will see a new menu button in the left side bar of your WordPress called 'Human Anatomy', click it and customize the diagram then save the changes.
4. Use this shortcode `[anatomy]` to display the diagram in any page or post.

== Frequently Asked Questions ==

= Can I add an image in the hover popup? =

Yes, you can write any text and add images through the visual editor in the dashboard.

= Does this diagram requires Flash to work? =

No, it's SVG based and works perfectly on desktop along with mobile devices 'smartphone'.

= Is it responsive? =

Yes.

= Can I use the shortcode in different pages? =

Yes, and that will display the same diagram.

= Can I link an organ or body part to another webpage? = 

Yes, and that page can be a separate page or a section in the same page.

= Can I set an organ/body part such as head as inactive? =

Yes, simply un-check the (active) box beside that organ/ body part to make it inactive.

= Do you offer special customization if needed? =

Yes, we can customize the diagram for what you need or create a new one for you as a freelance customization service, just drop us a line through this [contact](https://www.humananatomyillustrations.com/assets/contact.html) form.
